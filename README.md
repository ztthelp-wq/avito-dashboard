# Avito Pulse Dashboard

Node.js dashboard for Avito API metrics.

## Required environment variables

```env
AVITO_CLIENT_ID=
AVITO_CLIENT_SECRET=
AVITO_USER_ID=
AVITO_STATS_FROM=2025-01-01
PORT=4173
DASHBOARD_USER=
DASHBOARD_PASSWORD=
```

## Run locally

```bash
node server.js
```

## Deploy

Use `render.yaml` for Render Blueprint deploy, or `Dockerfile` for any Docker host.
